why

Glory! No, just kidding!...
You believe software and data interoperability is essential in furthering research and affecting positive outcomes for research. Both open dictionaries and open identifiers are critical pieces to solving the interoperability issues faced by systems that enable the research and scholarly communication ecosystem.

Codefest also are not just about deliverables, it’s one of the few times coders can socialize, learn about new ideas and new tools.

Awards (ok maybe a little bit of glory):

  • CASRAI - Awards to the best project that experiments with real use cases of automated machine-to-machine, API-based interoperability with Casrai Profile instances as 'payloads'. Examples might include funder-to-institutional cases like announcing funding results or institution-to-funder cases like submitting CV data to an application process or publisher-to-publisher cases like exchanging peer-review data. We intend explore possible reference implementations and play with JSON and/or XML and to feed back to the Casrai Tech Advisory Group on lessons learned and opportunities discovered.
  • ORCID - Sometimes organizations don’t have technical resources to interact with ORCID’s APIs directly. What tools can we create to enable these users to mint and manage ORCID record through administrative configuration, file uploads, or web-based services? These awards will go to the best project that address this need.
  • OPEN - You have an idea that doesn’t quite fit in either category? Well we are not going to get in the way of serendipity. Show up and code away! There will be awards to the best overall project.
